# Contributing

Thanks for your interest in contributing to Media Categories.

## Getting Started

1. Fork the repository and create a branch for your work.
2. Keep changes focused and easy to review.
3. Update documentation when behavior changes.
4. Add or update tests when practical.

## Development Notes

- Place the plugin in a local WordPress installation under:
  `wp-content/plugins/media-categories`
- Activate `Media Categories` from the WordPress admin before testing admin UI changes.

## Quality Expectations

- Follow WordPress coding standards.
- Prefer secure defaults, capability checks, nonce verification, and escaped output.
- Preserve accessibility in WordPress admin interfaces.
- Avoid unrelated refactors in feature or bug-fix pull requests.

## Pull Requests

Please include:

- a short summary of the change
- screenshots or notes for UI/admin changes
- testing notes

## Communication

For significant changes, opening an issue or starting a discussion first is appreciated.
